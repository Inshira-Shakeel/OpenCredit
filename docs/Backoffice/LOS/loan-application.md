## 📘 Loan Applications

The Loan Applications page is the operational hub for managing all loan requests.
It allows backoffice users to view, search, filter, and take actions on each loan application.

This page ensures you can quickly access details and monitor loan lifecycle status.

## 📋 Viewing Applications

The main interface displays all loan applications as records.
Key features:

* List of applications → Each row represents a loan request
* Click to view details → Access full loan data, including customer info and loan specifics
* Search input fields → Quickly find applications by customer name, ID, or other identifiers
* Filters → Narrow down by loan status, product type, or branch

## 💳 Application Details

Clicking a loan record opens a detailed view:

* Repayment information → See schedule, amount paid, balance remaining
* Loan status → Track approval, disbursement, or closure status
* Disbursement actions → If the loan is approved but not disbursed, authorized users can process disbursement directly from this page
